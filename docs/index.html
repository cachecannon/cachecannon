valkey-lab

A deep-dive benchmarking tool for Valkey and Redis. Go beyond quick throughput checks. Mix workloads, control rate, watch tail latency per second, and find your server's saturation point.

Get Started GitHub
Why a lab?

Go deeper than a quick benchmark

valkey-benchmark is a solid tool for quick throughput checks. valkey-lab builds on that foundation with controls for the questions that come next: what happens under mixed traffic, at a controlled rate, with realistic key distributions, over time?

Mixed GET/SET

Run GETs and SETs concurrently at any ratio (-r 80:20) to see how your server handles real contention patterns.

Rate limiting

Pin the load to a specific request rate (--rate-limit 500000) and measure latency under controlled conditions.

Prefill & warmup

--prefill populates the keyspace before benchmarking. --warmup discards early samples so you measure steady state.

Per-second tail latency

p50 through p9999 and max, every second. Spikes that hide in aggregate percentiles show up in the time series.

Key distribution

--distribution zipf models a realistic hot-key pattern. Uniform is the default; Zipfian reflects production skew.

Exportable results

--parquet results.parquet saves full time-series data. valkey-lab view opens an interactive web dashboard.

Saturation Search

Find max SLO-compliant throughput

Ramps request rate geometrically until your latency SLO breaks. Classifies failures as throughput-limited (server can't keep up) or latency-exceeded (server keeps up but tail latency blows). 

valkey-lab saturate --slo-p999 1ms -c 16 -P 32

Get Started

Install and run

# build from source (requires Linux)
cargo install --path . --bin valkey-lab

# benchmark localhost:6379, 60s, 80:20 GET:SET
valkey-lab

# 16 connections, pipeline 32, Zipfian keys
valkey-lab -c 16 -P 32 --distribution zipf

# prefill cache, then benchmark GETs only
valkey-lab --prefill -r 100:0

# measure latency at exactly 500K req/s
valkey-lab --rate-limit 500000 -c 16 -P 32

# find max SLO-compliant throughput
valkey-lab saturate --slo-p999 1ms

# save to parquet, view in browser
valkey-lab --parquet results.parquet -d 5m
valkey-lab view results.parquet

Flags follow valkey-benchmark conventions: -h, -p, -c, -P, -d, -t. Full reference in VALKEY-LAB.md.
